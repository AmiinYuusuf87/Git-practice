Git is the local software tool that tracks your code history offline.

GitHub is a cloud-based platform where you host and share Git repositories.

Git is like a camera taking snapshots; GitHub is like Instagram sharing them.

You can use Git entirely on your own computer without ever touching GitHub.

Teams use GitHub to collaborate, review code, and backup their local work online.

Alternatives to GitHub include other cloud hosting platforms like GitLab and Bitbucket.

Never say "I uploaded my code to Git" because Git is just the tool.

You track your code changes using Git, then upload them to GitHub.

To check if Git is installed on your machine, run the command git --version.

If Git is not installed, download the installer from the official git-scm.com website.

On macOS, you can easily install Git using the Homebrew command brew install git.

On Linux Debian/Ubuntu systems, install Git by running sudo apt install git.

You must configure your Git identity once before saving any local changes.
