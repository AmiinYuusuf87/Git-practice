#  Git & GitHub Bootcamp Project

This repository serves as a practical tracking tool for a foundational 3-lesson course in version control.

##  Course Overview & Lesson Summaries

 **Lesson  (Introduction to VCS):** Covers the historical challenges of manual file tracking and highlights the core structural advantages of Distributed Version Control Systems (DVCS) like Git.
**Lesson 1 (Git vs. GitHub & Config):** Differentiates between local machine tracking (Git) and cloud hosting collaboration platforms (GitHub), along with primary user environment configuration.
 **Lesson 2 (Core Workflow & Architecture):** Explains Git's three functional internal environments (Working Directory, Staging Area, and Local Repository) alongside hands-on execution of basic terminal commands.

## Technical Practice
* **Command Competency:** Development of operational hands-on speed and muscle memory using `git init`, `git status`, `git add`, and `git commit`.
